发表于 2018-06-06 | 分类于 Blog
在Centos7上搭建python（2+3）环境

Liunx+Python（2+3）部署

liunx系统自带python2版本，本文：

1、补充python3版本的部署

2、python2和python3版本共存及使用

一、Python3部署

1、安装软件依赖包

yum install openssl-devel bzip2-devel expat-devel gdbm-devel readline-devel sqlite-devel

2、上传python3源码包到指定目录，并解压–>进入解压后的目录

mkdir /home/installpackpage     #在home目录下创建指定目录，用于存放安装包
cd    /home/installpackpage     #进入安装包存放目录
tar   -zxvf   Python-3.6.2.tgz  #解压安装包
cd    Python-3.6.2                #进入解压后的python3源码目录

3、编译python3源码

./configure -prefix=/usr/local/python3 #在源码目录内，执行源码预编译操作，且指定安装python3的路径
sudo make && make install              #执行编译且编译安装操作，&&代表且，执行一个继续下一个
                                       #sudo命令临时获取root权限，root用户执行安装不需要
@ Successfully installed pip-9.0.1 setuptools-28.8.0 #成功后的提示

二、Python（2+3）部署

4、给python3及pip3创建软连接

ln -s /usr/local/python3/bin/python3 /usr/bin/python3 #为python3创建软连接
ln -s /usr/local/python3/bin/pip3 /usr/bin/pip3       #为pip3创建软连接

**注意：创建软连接的目的在于让系统可以找到直接调用python3和pip3的位置**
**       创建软连接必须使用绝对路径                                **
**       pip模块是python的模块管理工具，方便安装模块                 **

pip3 -V         #查看pip3的版本及对应的python版本
#pip 9.0.1 from /usr/local/python3/lib/python3.6/site-packages (python 3.6)
pip install bs4 #pip3针对python3安装第三方库
#beautifulSoup 是python支持的一个第三方的包,作用就是用来解析html网页,提取数据

5、给python2添加pip功能

yum -y install epel-release #为python2安装扩展源
yum install python-pip        #为python2安装pip

6、python初体验

@python（2+3）：
* pip  --->python   #python2版本
* pip3 --->python3  #python3版本
@python目录介绍：
* bin          #执行目录
* include     #python的库文件
* lib         #一些python模块的源码
* share          #帮助文档，开源协议
